# AI/ML From Scratch

A disciplined, first-principles journey through Python and AI/ML engineering, every line of code written by hand, no AI assistance. Using AI to learn and not to code. 

## Goal

To become an engineer who doesn't just use AI tools, but deeply understands the foundations beneath them, from basic Python to building ML models from scratch.

## Journey Roadmap

| Phase | Topics | Status |
|-------|--------|--------|
| Phase 1 | Python Foundations  | ✅ In Progress |
| Phase 2 | Problem Solving — logic building, patterns, clean code | 🔜 Upcoming |
| Phase 3 | Math for ML — NumPy, linear algebra, probability | 🔜 Upcoming |
| Phase 4 | ML From Scratch — algorithms built by hand | 🔜 Upcoming |
| Phase 5 | Deep Learning & AI Engineering | 🔜 Upcoming |

## Principles

- **Struggle first.** Every problem is attempted independently before seeking help.
- **Clean code always.** PEP 8, meaningful names, no shortcuts.
- **Understand, don't memorize.** Every line has a reason.
- **Build real things.** Not just exercises — real programs with real logic.
